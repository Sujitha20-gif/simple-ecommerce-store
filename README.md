# Simple E-commerce Store

This project is a basic full-stack e-commerce web application built as part of Task 1.

## Features
- Product listing
- Shopping cart
- Product details
- Order processing (basic)
- Backend API using Express.js

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js

## How to Run
1. Install Node.js
2. Run `npm install`
3. Start server using `node server.js`
4. Open `index.html` in browser
